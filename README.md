# 🚀 Portfólio Pessoal | Rodrigo da Costa Mendes

Bem-vindo ao meu espaço digital! Este projeto foi desenvolvido para consolidar minha trajetória profissional, unindo minha experiência prática em **Artefinalização e Design Gráfico** com minha atual jornada acadêmica em **Análise e Desenvolvimento de Sistemas (ADS)**.

## 🎯 Sobre o Projeto
Este portfólio serve como uma vitrine das minhas competências, demonstrando a transição da comunicação visual para a lógica de programação. O foco é apresentar soluções criativas aliadas a um código limpo e responsivo.

## 🛠 Tecnologias Utilizadas
O projeto foi construído com foco em performance e elegância visual:

* **HTML5:** Estruturação semântica.
* **CSS3:** Design responsivo, *Glassmorphism* e animações customizadas.
* **JavaScript:** Lógica de navegação dinâmica (Scroll Spy).
* **Font Awesome:** Ícones vetoriais.
* **Google Fonts:** Tipografia moderna (Inter e Open Sans).
